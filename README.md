Требования:

Вариант задания: 94. Условный оператор if-else на языке PHP

Примеры кода (многострочные в том числе)
1)
if ($a > $b) {
    $max = $a;
} else {
    $max = $b;

3)
if($a>$b){
    $a--;;
}
elseif($a < $b){
